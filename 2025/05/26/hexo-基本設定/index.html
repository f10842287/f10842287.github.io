Hexo 基本設定

初始化專案

hexo init [folder name]

範例

hexo init my-blog

建立新文章

hexo new [layout] <標題>

範例

hexo new post "第一篇文章"

local 端啟動伺服器

hexo server # 預設為 http://localhost:4000/

調整文章內容

當執行完以下指令後:

hexo new post "第一篇文章"

會在 source/_posts/ 底下產生 .md 檔案，source/_posts/第一篇文章.md
